*   **Key Tech**: C++11, Progressive Taxation Algorithms, MT19937 Randomization.

## Technical Standards

*   **Language**: Primary focus on C++ (Standard 11 and higher).
*   **Methodology**: High modularity, clean header management, and robust error handling.
*   **Design**: Low-level console rendering and efficient data processing.

## Development Principles

1.  **Code Integrity**: Every module must be isolated and independently testable.
2.  **Performance**: Minimal overhead with optimized logic flows.
3.  **Scalability**: Projects are designed to be expanded without breaking core functionality.
